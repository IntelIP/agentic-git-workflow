# Brand System

Tabellio uses a notary-orange visual system: near-black surfaces, off-white text, and one warm accent for attestation, provenance, and approval states.

## Logo

| Asset | Use |
| --- | --- |
| `docs/assets/tabellio-mark.svg` | Square mark, favicon, social avatar, compact UI |
| `docs/assets/tabellio-logo.svg` | Full horizontal logo lockup |
| `docs/assets/tabellio-hero.svg` | README and product overview hero |

The mark combines three ideas:

- a notarial seal for attestation
- a hidden `T` for Tabellio
- a connected path for Git provenance

## Color Stack

| Token | Hex | Use |
| --- | --- | --- |
| `base` | `#000000` | Page background |
| `surface` | `#111111` | Primary cards and logo field |
| `surface-muted` | `#191919` | Secondary panels and inputs |
| `border` | `#3A3A3A` | Rules, dividers, inactive outlines |
| `accent` | `#DA5C2C` | Attestation paths, active states, primary emphasis |
| `accent-soft` | `#F2B37B` | Node highlights and small warm details |
| `text` | `#EEEEEE` | Headings and logo text |
| `text-muted` | `#B4B4B4` | Supporting copy |

## Usage Rules

- Use orange as signal, not decoration.
- Keep surfaces mostly black and graphite.
- Prefer vector SVG for logos; export PNG only for compatibility.
- Keep radius low for product UI and cards.
- Avoid blue/teal gradients in Tabellio brand assets.
